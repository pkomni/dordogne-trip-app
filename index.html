Persistence
• Local only: All edits stay on this device (localStorage).
• GitHub Pages: Just commit this file and enable Pages—no backend needed.
• Snapshots: Use the FAB menu to save and restore versions when you want.
How to use
  • 1. Pick a day from the chips at the top.
  • 2. Tap “✏️ Edit” on an activity or restaurant to open the bottom sheet editor.
  • 3. Use the “+” FAB to add items, save snapshots, or reset to default.
  • 4. Use “Map today’s restaurants” to open Google Maps with today’s picks.
