💾 Persistence
• Local only: All edits are stored in your browser (localStorage).
• GitHub Pages: Commit this file and enable Pages—no backend needed.
• Snapshots: Use the floating “+” menu to save and restore versions.
✨ How to use
1. Slide the day row and tap a day card.
2. Tap “✏️ Edit” on an activity or restaurant to adjust details.
3. Use the “+” button (bottom‑right) to add items, save snapshots, or reset.
4. Use “Map today’s restaurants” to open Google Maps with today’s picks.
💡 Tips
• Keep it light: you don’t need to fill every field.
• Use snapshots before big changes, so you can roll back easily.
• Treat this like a personal travel app—simple, visual, and forgiving.
